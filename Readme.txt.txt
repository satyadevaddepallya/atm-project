Prerequisites
1- Install Java 8 or higher
2- Install Gradle 5.x or later


Build Instructions

1. Unzip the attached file to a location in computer
2. Go to folder - atm-project
3. run gradle clean build.
4. Once build successful run gradle fatJar
5. then run executable jar generated in atm-project/build/libs/atm-project-all-1.0.jar
java -jar atm-project-all-1.0.jar
